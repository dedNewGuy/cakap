# Cakap

Basic IRC Client implemented in C for the Terminal.

> [!WARNING]
> This project is still under construction. It's fairly usable but
> very barebone.

## Build
**Requirements**
- make
- unix

```bash
make
./cakap erbium.libera.chat nickname123 username456
```
